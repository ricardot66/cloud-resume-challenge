About This Website

Made with Cloud in Mind.

This website is hosted on Amazon Web Services (AWS), leveraging a modern and scalable cloud architecture and meant as an cloud computing exercise based on The Cloud Resume Challenge Guidebook by Forrest Brazeal. Here's a peek behind the scenes:
Static Hosting: The site is hosted using Amazon S3, ensuring fast, reliable, and cost-effective delivery of static content.
Content Delivery: To improve global performance and reduce latency, CloudFront serves as the Content Delivery Network (CDN), caching assets closer to visitors.
Secure Browsing: The site uses SSL encryption to protect your connection, ensuring secure data transmission.
Custom Domain: The domain is managed via GoDaddy, configured to point to the CloudFront distribution.
Visitor Counter: A simple and serverless visitor counter tracks visits in real time.
Serverless: A Lambda function that updates and retrieves visit counts. API Gateway acting as the interface between the website and backend services. A DynamoDB table that stores the visitor count data. This architecture enables high performance, scalability, and a fully serverless backend—optimized for both developers and visitors alike.
